joyful linux

apr 21, 2019

as part of a procrastination effort from having to study for final exams, i've started working on joyful linux, a personal exploration of building an experimental, nonstandard, and personal linux-based system. in essence, it is to start from nearly nothing and then patiently add only the things that spark joy.

the joyful in joyful linux is a reference to marie kondo, whose writings and philosophy have helped shape my own lifestyle, both in the real and virtual world.

for now, all relevant repositories can be found here.

i am also maintaining a detailed development log.
