joyful linux

a personal exploration of building an experimental, nonstandard, personal linux-based system. it is to start from nearly nothing and then patiently add only the things that spark joy.

i have briefly glossed over lfs, and found it to be too high-level and rigid. i want to truly start from scratch and figure everything out myself.

when joyful linux reaches 0.1, it might be a distribution by then, and this will become a home page for it. until then, this is home to an informal development log.
